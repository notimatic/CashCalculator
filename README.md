# Cash Calculator

An easy to use command-line tool written in C# that quickly calculates amount of money spent on itemized list of things. Each amount being added can be marked to include or exclude VAT (20%) in calculation. The items are entered in a row and separated with spaces. The results are saved in a text file.

Expense Calculator CLI Tool

A simple and efficient command-line utility written in C# for calculating total expenses from an itemized list. Each item can be flagged to either include or exclude VAT (20%) during the calculation.

- Input Format: Items are entered in a single line, separated by spaces.

- VAT Handling: Each amount can be marked to include or exclude VAT.

- Output: The final results are saved to a text file for easy reference.

Perfect for quick budgeting, invoice checks, or expense tracking directly from the terminal.
